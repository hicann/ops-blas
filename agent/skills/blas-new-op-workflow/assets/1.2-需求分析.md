## 3. 开发约束

### 3.1 参数约束

Host 侧需在调用 Kernel 前完成参数合法性校验：

| 条件 | 返回值 |
|------|--------|
| `handle == nullptr` | `ACLBLAS_STATUS_NOT_INITIALIZED` |

### 3.2 编码约束

- 禁止使用 `SetValue`/`GetValue`，数据搬运必须使用 `DataCopy`/`DataCopyPad`
- 禁止对单一元素进行 `DataCopy`/`DataCopyPad` 操作
- 使用 `aclrtGetDeviceInfo` 动态获取 CoreNum，禁止硬编码核数
- `TPipe` 禁止作为类成员变量，应在 Kernel 函数入口创建
